# Java University Portfolio — Monmouth University

This repository contains academic projects and lab exercises developed during my **Computer Science** studies at **Monmouth University**. It showcases proficiency in Java development, object-oriented programming (OOP) principles, and data structure implementations.

## 🏗 Project Structure
The source code is organized by course modules following the standard Java package naming conventions:

* **`edu.monmouth.CS205`**: **Intermediate Programming & Algorithms**. Focuses on theoretical concepts, algorithm efficiency, and advanced logic.
* **`edu.monmouth.CS205L`**: **Programming Lab**. Hands-on implementation of Java concepts, including class design, inheritance, and modular programming.

## 🛠 Tech Stack
* **Language:** Java 17+
* **IDE:** Eclipse / VS Code
* **Architecture:** Modular Java (`module-info.java` integrated)
* **Version Control:** Git/GitHub

## 🚀 Key Features
* **Modular Design:** Leveraging the Java Module System for better encapsulation and security.
* **OOP Principles:** Practical application of Inheritance, Polymorphism, and Encapsulation.
* **Clean Code:** Organized directory structure following industry-standard scalability and readability.
